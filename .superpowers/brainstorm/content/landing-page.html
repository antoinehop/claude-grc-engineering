What it does
The full GRC engineering pipeline,
shipped as Claude Code commands
Every step from evidence collection to remediation — in your terminal, backed by an open data contract any connector can plug into.
◈
Gap Assessment
Pull findings from cloud and SaaS connectors, crosswalk to any framework via SCF, and get a prioritized, effort-estimated gap report.
/grc-engineer:gap-assessment
⌥
IaC Scanning
Scan Terraform, CloudFormation, and Kubernetes for compliance violations. Auto-fix 47% of findings in one command.
/grc-engineer:scan-iac
✦
Code Generation
Generate production-ready Terraform modules, Python evidence collectors, and Rego/Cedar policies from any control ID.
/grc-engineer:generate-implementation
⊟
Cross-Framework Intelligence
Map one control to every framework it touches. Find conflicts, resolve them, and implement once to satisfy many.
/grc-engineer:map-controls-unified
◷
Continuous Monitoring
Schedule daily control tests, track 30-day compliance trends, and get Slack or PagerDuty alerts before auditors do.
/grc-engineer:monitor-continuous
⊞
OSCAL Output
Generate OSCAL SSP, SAP, SAR, and POA&M from findings and framework configs. FedRAMP-ready artifacts in one command.
/oscal:ssp-export
